# 🎯 Website Dev Skill - 网站开发技能包

> 宣传网站从规划到上线的完整解决方案

## 📦 包含内容

| 模块 | 说明 |
|------|------|
| HTML/CSS/JS | 编码规范 + 模板 |
| SEO | meta、结构化数据 |
| 无障碍 | WCAG 2.1 AA |
| 性能 | Core Web Vitals |
| 动画 | Animate.css / AOS / GSAP |
| 组件 | Tailwind CSS + DaisyUI |
| 测试 | 自动化功能测试 |

## 🚀 快速开始

### 1. 复制 SKILL.md

将 `SKILL.md` 复制到你的 Agent Skills 目录：

```
skills/website-dev/SKILL.md
```

### 2. 技术栈选择

| 类型 | 推荐方案 |
|------|---------|
| 简单宣传站 | HTML + CSS + JS |
| 中型商业站 | Vue 3 + Vite + Tailwind |
| 企业站 | Vue/React + 组件库 |

### 3. 开发流程

```
需求确认 → 技术方案 → 编码实现 → 自检测试 → 交付上线
```
